sales Analysis Project (SQL)

This project focuses on analyzing sales data to identify top-performing regions and the top 10 buyers by revenue. The analysis was done entirely using SQL, with an emphasis on transforming raw data into meaningful business insights.

 Skills & Concepts Applied

SQL Aggregations: GROUP BY, SUM, COUNT

Window Functions: PARTITION BY, ORDER BY

Ranking & Sorting: Identifying top regions and highest-revenue buyers

Data Analysis: Revenue-based insights and performance comparison

...Key Insights

Top sales regions based on total revenue

Top 10 buyers contributing the highest revenue

Clear ranking and ordering of results for decision-making

...Visuals

PNG images are included to visually present the results and support the SQL findings.
